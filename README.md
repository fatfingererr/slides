# Slides

自動生成的簡報集合，由 [content-to-slides](https://github.com/fatfingererr/slides) 產生。

## 🌐 GitHub Pages

訪問: https://fatfingererr.github.io/slides/

## 📁 結構

```
.
├── index.html          # 入口頁面，列出所有簡報
├── *.html              # 個別簡報檔案
└── push.sh             # 推送腳本
```

## 🚀 新增簡報

1. 產生新簡報到這個目錄
2. 執行 `./push.sh` 自動推送

## 📝 Commit Message 規則

所有自動推送使用: `feat: add new slides`
